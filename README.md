# Tres en Raya (Tic-Tac-Toe con Pygame)

Un sencillo pero vistoso juego de Tres en Raya (Tic-Tac-Toe) local para dos jugadores, desarrollado en Python con Pygame. ¡Juega por turnos, disfruta de una interfaz moderna y reinicia fácilmente la partida!

## Descripción

Tres en Raya es un clásico juego de tablero para dos personas. En este proyecto podrás:
- Jugar localmente con otra persona usando el ratón.
- Disfrutar una interfaz limpia, con colores vibrantes y textos en español.
- Ver mensajes automáticos al ganar, empatar o reiniciar.

## Instalación y uso

1. **Crea un entorno virtual:**

   ```
   python -m venv venv
   ```

2. **Activa el entorno virtual:**

   - En Windows:
     ```
     venv\Scripts\activate
     ```
   - En macOS/Linux:
     ```
     source venv/bin/activate
     ```

3. **Instala las dependencias:**

   ```
   pip install -r requirements.txt
   ```

4. **Ejecuta el juego:**

   ```
   python main.py
   ```

## Controles

- Haz clic en una casilla vacía para colocar tu ficha ("X" o "O"), alternando por turnos.
- Cuando alguien gana o hay empate, pulsa la tecla `R` para reiniciar la partida.
